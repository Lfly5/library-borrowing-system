# 📚 Library Borrowing System - Group 25

## 📖 Project Overview
This project is a functional simulation of a Library Borrowing System, developed as the practical deliverable for **COMP 3510SEF / COMP S351F: Software Project Management

The system automates manual record-keeping, provides real-time inventory updates, and offers an intuitive interface for students to search, borrow, and return books

## ✨ Core Features
**Book Inventory Search:** Intuitive search interface to find available books by title or author.
**Borrow & Return Processing:** Streamlined tracking of book circulation.
**Real-time Updates:** Automatic inventory adjustment upon borrowing or returning.
**System Activity Log:** Automated record-keeping to track user actions.

## 🚀 Installation & Setup Guide
This application is built as a lightweight, front-end only simulation using HTML, CSS, and JavaScript. 

**No backend installation, database configuration, or specific programming environment is required.**

### How to Run the Application:
1. Access the repository via the provided GitHub link.
2. Click the green **"Code"** button and select **"Download ZIP"**.
3. Extract the downloaded ZIP folder to your local computer.
4. Double-click the `index.html` file. It will automatically open in your default web browser (e.g., Chrome, Edge, Safari).
5. The system is ready to use! You can now interact with the interface to test the features.
